# Image Assets

This directory contains all images for the Moroccan Qamis website.

## Required Images

### Hero Section
- `hero/qamis.jpg` - Main hero background image (1920x1080px recommended)

### Products
Add product images named:
- `products/qamis-1.jpg` through `products/qamis-8.jpg`
- Recommended size: 800x1200px (3:4 aspect ratio)
- Format: JPG or WebP for best performance

### Categories/Patterns
- `patterns/classic.jpg` - Classic qamis category image
- `patterns/embroidered.jpg` - Embroidered category image
- `patterns/modern.jpg` - Modern cuts category image
- `patterns/luxury.jpg` - Luxury collection category image
- Recommended size: 600x600px (square)

### Icons
- Optional: Add custom icons or logo images here

## Image Optimization Tips

1. **Compress images** before uploading:
   - Use tools like TinyPNG, ImageOptim, or Squoosh
   - Target: < 200KB per image

2. **Use WebP format** for modern browsers:
   - Better compression than JPG
   - Supported by all major browsers

3. **Provide alt text** in HTML for accessibility

4. **Lazy loading** is already implemented in the code

## Placeholder Images

If you don't have images yet, you can use placeholder services:
- https://placeholder.com/
- https://via.placeholder.com/
- https://picsum.photos/

Example:
```html
<img src="https://via.placeholder.com/800x1200/2A7C5B/FFFFFF?text=Qamis+1" alt="Product">
```
